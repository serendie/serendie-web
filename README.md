# Serendie Website

Cloudflare Pagesでホスティングしています。

## ステージング環境へのデプロイ

URL: https://dev.serendie-web.pages.dev/ (要Basic認証)

`dev`ブランチにpushすると、ステージング環境へデプロイされます。

## プロダクション環境へのデプロイ

URL: https://serendie.design/

`main`ブランチにpushすると、プロダクション環境へデプロイされます。

## ローカルでの開発

```
npm install
npm run dev
```
