# Serendie Design System Website

## Staging

URL: https://dev.serendie-web.pages.dev/

`dev`ブランチにpushすると、ステージング環境へデプロイされます。

## Production

URL: https://serendie.design/

`main`ブランチにpushすると、プロダクション環境へデプロイされます。

## ローカルでの開発

```
npm install
npm run dev
```
