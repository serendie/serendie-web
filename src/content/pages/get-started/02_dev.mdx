    <PageHeadLine number={1}>デザインを確認する</PageHeadLine>

    ### Devモード

    三菱電機のFigma組織内では、開発者向けにDevモードアカウントを準備しています。Devモードは開発者向けのFigmaビューであり、実装時のヒントとなるインスペクタやアノテーションを活用できます。詳しくはFigmaヘルプページを参照してください。

    <PageARef href="https://help.figma.com/hc/ja/articles/15023124644247-%E9%96%8B%E7%99%BA%E3%83%A2%E3%83%BC%E3%83%89%E3%81%AE%E3%82%AC%E3%82%A4%E3%83%89">Devモードのガイド - Figma Learn</PageARef>

    ### Code Connect

    Serendie UIのコンポーネントはCode Connectに対応しています。Code Connectは、デザインファイル上のコンポーネントに対応するコードスニペットをFigma Devモードに表示する機能です。Code Connectを活用することで、デザインを確認しながら、Reactコンポーネントの実装例を同時に確認することができます。

    <p>
      <img src="/assets/pages/code-connect.png" width="310" alt="" />
    </p>

    ### Storybook

    またCode Connectと同等の内容を、Storybookとして提供しています。UIコンポーネントの状態変化や、ReactコンポーネントのPropsとの対応など、より詳しく確認したい場合に利用できます。


    <PageARef href="https://serendie.design/storybook">Storybook - Serendie Design System</PageARef>

  </PageSection>

  <PageSection>
    <PageHeadLine number={2}>開発の準備</PageHeadLine>

    ### GitHubチームに加わる

    GitHubの[Serendieリポジトリ]()は公開に向けて準備中です。現在のところプライベート状態で運用しているため、三菱電機社内で利用希望の方は、Serendie Design Systemチームに連絡しGitHubチームへの招待を受けてください。

    ### GitHub Packagesの利用準備

    Serendie UIはnpmパッケージとして利用可能です。現在のところレジストラとして、GitHub Packagesを利用しており、事前にアクセストークンの準備が必要です。詳しくは[README]()を参照してください。

  </PageSection>

  <PageSection>
    <PageHeadLine number={3}>デザイントークンを使う</PageHeadLine>

    Serendie UIとデザイントークンは別のパッケージとなっており、独立して利用可能です。デザイントークンは、CSS, JavaScript環境であれば広く利用可能な形で配布しています。npmなどのパッケージマネージャーで下記のようにインストールしてください。なお、デザイントークンの一覧は[Foundations/Design Tokens]()を参照してください。

    <CodeBox code={`npm install @serendie/design-token`} language={'shell'} client:idle />

  </PageSection>

  <PageSection>
    <PageHeadLine number={4}>Reactコンポーネントを使う</PageHeadLine>

    Serendie UIはReact環境に対応しています。Reactコンポーネントを利用したい場合は、下記のようにインストールしてください。

    <CodeBox code={`npm install @serendie/ui`} language={'shell'} client:idle />

    また、各プロジェクトのルートCSSファイルにおいて、下記のようにSerendie UIのCSSを適用してください。

    <CodeBox code={sampleCodeCss} language={'css'} client:idle />

    そのうえで、利用したいUIコンポーネントをimportしてプロダクトを実装してください。各種UIコンポーネントの詳細やサンプルコードは、[Components]()を参照してください。

    <CodeBox code={`import { Button } from "@serendie/ui";`} language={'jsx'} client:idle />

    ### APIを詳しく知る

    Serendie UIはヘッドレスUIライブラリとして[Ark UI](https://ark-ui.com/)を内部的に利用しており、各コンポーネントのAPIはArk UIを継承します。Selectコンポーネントなどインタラクションが複雑なコンポーネントは、Ark UIのAPIリファレンスを合わせて参照してください。

    <PageARef href="https://ark-ui.com/react/docs/components/select#api-reference">Select | API Reference - Ark UI</PageARef>


    ### カスタマイズして使う

    Serendie UIはスタイリングライブラリとして[Panda CSS](https://panda-css.com/)を利用しています。Panda CSSが提供するユーティリティを使ったコンポーネントの拡張や、テーマの変更などが可能です。詳しくはREADMEを参照してください。なお、カスタマイズせずに使う場合は、Panda CSSの導入は必要ありません。
