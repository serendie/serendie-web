このページでは、MCPサーバーの機能をインタラクティブに試すことができます。実際にMCPツールを呼び出して、レスポンスを確認できます。

## 使い方

1. 下のデモセクションで試したいツールを選択
2. 必要なパラメータを入力
3. 「実行」ボタンをクリック
4. レスポンスを確認

**注意**: このデモを使用するには、開発サーバーが起動している必要があります（`npm run dev`）。

## シンボル検索の例

`get-symbols`ツールを使用して、特定のキーワードでシンボルを検索：

  <CodeBox code={`// 検索パラメータ
{
  "search": "arrow",
  "limit": 10
}

// レスポンス例
{
"totalCount": 150,
"filteredCount": 8,
"returnedCount": 8,
"availableVariants": ["outlined", "filled"],
"symbols": [
"arrow_back",
"arrow_forward",
"arrow_downward",
"arrow_upward",
"arrow_drop_down",
"arrow_drop_up",
"arrow_left",
"arrow_right"
]
}`} language="json" />

## デザイントークン取得の例

`get-design-tokens`ツールを使用して、カラートークンを取得：

  <CodeBox code={`// 検索パラメータ
{
  "type": "color",
  "category": "system",
  "limit": 5
}

// レスポンス例
{
"totalCount": 450,
"filteredCount": 120,
"returnedCount": 5,
"availableTypes": ["color", "typography", "dimension", "elevation", "radius", "spacing", "opacity"],
"tokens": [
{
"key": "sd.system.color.primary",
"path": ["sd", "system", "color", "primary"],
"type": "color",
"value": "#0066CC",
"originalValue": "{sd.reference.color.blue.60}",
"category": "system"
},
// ... 他のトークン
]
}`} language="json" />
