   <PageHeadLine number={1}>Figmaの基礎を知る</PageHeadLine>

    UIコンポーネントとデザイントークンをまとめたSerendie UI KitをFigma向けに提供しています。FigmaはUIデザインにおいてスタンダードになりつつあるツールであり、一枚絵としてデザインするのではなく、コンポーネントを組み合わせ構造的にデザインできる点が特に優れています。

    コンポーネント、バリアント、オートレイアウト、VariablesはFigmaの特徴的な機能であり、これらが備わっていることがUIデザインにおいてFigmaを使う大きな理由です。Serendie UI Kitもこれらの機能を前提に設計されています。Figmaのヘルプ記事を参照しつつ、まずは基礎を抑えてみましょう。

<PageSection>
    <PageHeadLine number={2}>Serendie UI Kitを使う</PageHeadLine>

    ### Figmaファイルにライブラリを加える

    三菱電機のFigma組織内では、ライブラリとしてSerendie UI Kitが利用可能な状態です。Serendie UI Kitを使いたいFigmaファイルから、ライブラリを検索し追加してください。社外の方向けには、Figma Communityにて最新のSerendie UI Kitを公開していますので、こちらを利用してください。

    <p>
      <PageARef href="">Serendie UI Kit - Figma Community</PageARef>
    </p>

    <p>
      <img src="/assets/pages/getstarted-design-1.png" width="310" alt="" />
    </p>

    ### デザイントークンを使う

    Serendie UI Kitをライブラリとして追加した後は、カラー、タイポグラフィ、スペーシングなどのデザイントークンをFigmaファイル上で利用可能です。デザイントークンについての詳しい解説は、[Foundations]()を参照してください。

    <p>
      <img src="/assets/pages/getstarted-design-2.png" width="310" alt="" />
    </p>

    ### UIコンポーネントを使う

    Serendie UI Kitが提供するUIコンポーネントは、Assetsパネルから利用できます。各UIコンポーネントを追加する際には、適切なバリアントを選択してください。UIコンポーネント及びバリアントの解説は、[Components]()を参照してください。

    <p>
      <img src="/assets/pages/getstarted-design-3.png" width="310" alt="" />
    </p>

    ### テンプレートを使う

    Serendie UI Kitでは、抽象度の高いUIコンポーネントだけでなく、それらを組み合わせた画面デザイン例 (テンプレート) を提供しています。ログイン画面やウォークスルー画面など、業務アプリケーションでよく登場するデザイン例をSerendie UIを使いつつ作成しているので、これを土台にしてデザインをスタートすることも可能です。

    <p>
      <PageARef href="">Templates - Serendie UI Kit</PageARef>
    </p>

      <img src="/assets/pages/getstarted-design-4.png" width="310" alt="" />
    </p>
