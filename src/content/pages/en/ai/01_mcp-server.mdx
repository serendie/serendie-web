<PageSection>
  <PageHeadLine number={0}>What is Serendie MCP Server</PageHeadLine>

The Serendie MCP Server is a <a href="https://modelcontextprotocol.io/introduction" target="_blank">Model Context Protocol</a> server designed to efficiently leverage the Serendie Design System from AI agents.

By connecting this MCP server to various AI agents, the AI can autonomously obtain guidelines, best practices, and resource lists—including components and icons—provided by the Serendie Design System. This enables an application development workflow that follows the Serendie Design System.

The Serendie MCP Server is published as a remote MCP server (Streamable HTTP Transport), so you can start using it immediately without any local setup by specifying the following endpoint.

<CodeBox code={`https://serendie.design/mcp`} language={"shell"} />

</PageSection>

<PageSection>
  <PageHeadLine number={1}>Toolset</PageHeadLine>

The MCP server provides capabilities to AI agents in units called "tools." The AI agent decides which tool to use based on the prompt and context. The tools currently provided by the Serendie MCP Server are:

- Overview tool for retrieving Serendie UI setup instructions and general information
- Tools for listing and retrieving details of Serendie UI components and their properties
- Tools for listing and retrieving design token details
- Tools for listing and retrieving Serendie Symbols details
- Tool for retrieving various guidelines published on this site

The implementation of each tool is available on the GitHub repository (serendie/serendie-web).

</PageSection>

  <PageHeadLine number={2}>Use Cases</PageHeadLine>

## Prototyping with Natural Language

Agentic coding tools such as [Claude Code](https://docs.claude.com/en/docs/claude-code/overview) and [Figma Make](https://www.figma.com/make/) enable prototyping with natural language. By adding the Serendie MCP Server, you can quickly build implementations using Serendie UI.

The Serendie MCP Overview tool provides an introduction to Serendie UI, allowing AI agents to autonomously set up the environment for using Serendie UI. Additionally, by using the various resource retrieval tools, AI agents can efficiently acquire the knowledge needed to use Serendie UI. This leads to higher-quality prototyping in agentic coding.

## Implementing from Designs

Serendie UI provides Figma components (design) and React components (implementation) as a pair, with mappings established through <a href="https://help.figma.com/hc/en-us/articles/23920389749655-Code-Connect" target="_blank">Figma Code Connect</a>. By combining the Serendie MCP Server with the <a href="https://help.figma.com/hc/en-us/articles/32132100833559-Guide-to-the-Figma-MCP-server" target="_blank">Figma MCP Server</a>, you can implement designs created with the <a href="https://serendie.design/en/get-started/design/" target="_blank">Serendie UI Kit</a> quickly and with high quality.

Using the Figma MCP Server, AI agents can read design data. After configuring the MCP server, simply tell your AI agent something like **"Implement the currently selected Figma design using Serendie UI"**—the AI agent will autonomously set up Serendie UI, learn how to use the various components, and generate high-quality code suited to your project environment.

  <PageHeadLine number={3}>Setup</PageHeadLine>

## Claude Code

Enter the following command in your terminal:

<CodeBox
  code={`claude mcp add --transport http serendie-mcp https://serendie.design/mcp`}
  language={"shell"}
/>

Click the link below to automatically add the MCP server.

## Visual Studio Code / GitHub Copilot

Open the <a href="https://docs.github.com/en/copilot/how-tos/provide-context/use-mcp/extend-copilot-chat-with-mcp" target="\_blank">MCP settings from the Command Palette</a> in Visual Studio Code, and add the following to your mcp.json:

<CodeBox
  code={`{
    "inputs": [],
    "servers": {
      "serendie-mcp": {
        "url": "https://serendie.design/mcp",
        "type": "http"
      }
    }
}`}
  language={"json"}
/>

## Figma Make

In Figma Make, workspace administrators can set up custom connectors to use the Serendie MCP Server. Specify the endpoint **https://serendie.design/mcp** in the custom connector settings.

<PageSection>
  <PageHeadLine number={4}>LLMs.txt</PageHeadLine>

LLMs.txt is a file that helps AI agents efficiently use the various Serendie UI documentation. In scenarios where an MCP server is not available, you can point your AI agent to this LLMs.txt to provide it with information equivalent to the guidelines on this site.

We provide two versions via the links below. Specify these URLs to your AI agent:

- [LLMs.txt](https://serendie.design/llms.txt) -- A concise list of document descriptions and URLs
- [llms-full.txt](https://serendie.design/llms-full.txt) -- Full text of all documentation
