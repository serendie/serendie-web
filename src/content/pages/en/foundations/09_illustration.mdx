  <PageHeadLine number={1}>Illustration system</PageHeadLine>

For the Serendie Design System operations team, we provide an easy way to create Serendie-style illustrations when making new documents. Access it from the Illustration page in the <a href={siteConfig.serendieUIKitFigmaFileURL}>Serendie UI Kit (Figma)</a>.

Currently, this is offered internally to the operations team. It is included in the Figma file but not published as a library (it is not available from the Assets panel inside Figma).

## 3. Confirm theme support

Colors in the illustrations are managed by Figma Variables. Similar to [UI theming](/en/foundations/theming#section-3), setting the variable mode automatically applies colors for each theme. Set the variable mode and verify colors per theme.
